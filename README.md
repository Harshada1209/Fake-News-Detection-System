# Fake-News-Detection-System
The saved folder will look like:

FakeNewsProject/
│
├── app.py                 # Streamlit UI file (main file to run)
├── project.ipynb          # Model training notebook
│
├── fake_news_model.pkl    # Trained model
├── svm_model.pkl          # SVM trained model (optional)
├── w2v_model.pkl          # Word2Vec model
├── word2vec_model.pkl     # Word vector file (if used)
│
├── Fake.xlsx              # Fake news dataset
├── True.xlsx              # True news dataset
│
└── README.txt (optional)  # Project description for submission
